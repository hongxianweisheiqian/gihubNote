起步 ​

基本配置 ​

捕捉模式 - HDV：表示高清格式

新建序列 ​

素材剪辑必须先创建序列。序列大小决定了输出视频的尺寸大小

步骤和基本配置 ​

文件 = > 新建 = > 序列

一般选择 HDV 1080p25 ，含义是高清 1080p 25 帧

带 i 的（如 HDV 1080i 25）一般是电视台使用的。24 帧一般是电影的帧数

也可以自定义序列，可以点击下方 “保存预设” 方便下次直接使用

注意 ​

当导入的视频和自己现有的序列不匹配时尽量继续保持现有设置

导出工程 ​

功能：当我们需要在另一台电脑上继续修改未完成的视频时，可以执行此操作

操作：视图左上角“文件”=>“项目管理” 即可打开导出界面，注意下图红色圈出的选项按需打勾，勾上后导出的工程文件中不会包含此刻工程里没有用到的素材

导出视频 ​

功能：当我们完成剪辑后就可以输出剪辑过的视频

操作：视图左上角“文件”=>“导出”=>“媒体” 即可打开导出页面，也可以使用快捷键 “ctrl”+“M” 快速导出

导出片段 ​

可以在时间轴面板的时间轴上按“i”打上入点标记再按“o”打上出点标记，再按 ctrl+m 键直接导出这一段视频。具体可翻阅本笔记 “面板” => "时间轴面板" => "导出片段和时间轴颜色"

导出音频 ​

功能：当我们完成剪辑后就可以输出剪辑过的音频

操作：视图左上角“文件”=>“导出”=>“媒体” 即可打开导出页面，也可以使用快捷键 “ctrl”+“M” 快速导出

面板 ​

项目面板 ​

调整图层 ​

当需要给所有视频片段统一调整样式可以使用

“调整图层” => “将新建出来的图层拖拽到序列栏所有需要修改样式的视频片段之上” => “切换顶部到颜色栏” => “单击序列栏里新建的图层” => “修改样式”

这样就可以按需修改视频样式

添加彩条 ​

pr 中默认就有彩条可直接使用

黑场视频 ​

顾名思义就是一段黑屏，可以对其修改样式，可以利用它做出过渡效果

源面板 ​

拖动视（音）频 ​

当鼠标左键按住此按钮拖动到时间轴面板时只会展示视频，即去音频

旁边拖动音频按钮同理

视频调整 ​

左边是调整视频画布的大小，即视频画布在源面板展示的大小

右边则是调整视频展示的分辨率

出入点 ​

如果只想取素材视频中间的某一段剪辑，则可以在源面板时间轴上点击一下并且按下键盘 “i” 键就会打上入点标记，同样操作按下键盘 “o” 键就会打上出点标记表示我们只需要这一段进行剪辑。（可将打上出入点的视频拖入到时间轴面板即可开始编辑）

标记点 ​

当我们需要视频某一帧画面做其他操作时可以将时间轴拖动到对应位置点击下方“添加标记”按钮，上方绿色的指针就表示刚刚打上的标记点，还可以双击标记点进行编辑

导出帧 ​

如果视频中某一帧的画面我们像保存下来当素材可以点击此按钮，功能就类似于截图

效果控件面板 ​

点击对应效果前方的 “fx” 代表开启/关闭 此效果

运动 ​

顾名思义 “运动、缩放、旋转” 就是改变原视频位置、缩放大小、旋转角度，也可以直接点击 “运动” 在右侧节目面板直接鼠标操作

锚点：会根据当前视频锚点作为中心点进行旋转，也可以修改锚点位置

不透明度 ​

可以在视频上加一层蒙版做一些效果，第一行三个图形（椭圆、方形、钢笔）是用来规定蒙版展示区域的形状的

蒙版路径

蒙版羽化：调节蒙版展示区域边缘的羽化值

蒙版不透明度：调节蒙版展示区域的不透明度

蒙版扩展：调节蒙版展示区域的大小，反转表示展示除蒙版区域以外的区域

不透明度：调节整个视频整体的不透明度

混合模式：当有多个视频素材处在同一时间轴上时可以通过下拉选项改变这些视频的叠加效果

声像器 ​

调节左（右）声道大小

音频剪辑混合器 ​

可以调节选中音频素材的声音

操作：下方红框中的 “写关键帧”（三个图标最右侧）可以在播放时拖动下方滑块调节音量大小

功能：实现视频在播放过程中音量随之变化

音轨混合器 ​

功能：可以控制某一轨道的所有音频

操作：在 Pr 软件顶部菜单栏选择 “窗口”，再点击 “音轨混合器” 即可开启此面板

如下图，点击红框中 “R” 图标再点击下方红框中的红色圆形图标即可开始录制画外音，空格键操作暂停/开始

音频小知识 ​

时间轴面板 ​

轨道详细信息 ​

双击轨道前端空白区域即可查看对应详细信息。如图双击音频轨道可查看右侧音频音量分布

控制视频透明度以及音量大小 ​

操作：点击扳手图标 => 显示视频关键帧

功能：视频素材里会有一根线，上下拖动它可以调节视频透明度。音频素材里会有一根线，上下拖动它可以调节音量大小

缩放为帧大小 ​

当素材分辨率和我们设置的序列不匹配时，可在对应素材上 “鼠标右键”=>"缩放为帧大小"，使素材自动匹配序列

轨道工具 ​

音频轨道上的 “M” 是静音轨道控制这个轨道有无声音

音频轨道上的 “S” 是当有多个音频轨道上下并排一起时，那么这一段可能会有多个声音（有多个音频轨道在这里），这时给哪个音频轨道点亮 “S” 那么之一段只会播放这条音频轨道的声音其他轨道会被屏蔽掉，简单说就是谁有“s”谁有声音

麦克风图标是画外音功能，就是点亮图标后可以直接对着麦克风录制声音

导出片段和时间轴颜色 ​

可以在时间轴上按“i”打上入点标记再按“o”打上出点标记，再按 ctrl+m 键直接导出这一段视频

时间轴上的颜色条（下图时间轴下方黄色长条）会根据当前电脑配置展示不同颜色：

黄色：正常 红色：卡顿 绿色：流畅

嵌套 ​

操作：在时间轴面板上选中多个视频 “右键” => “嵌套”

功能：可以将多个素材合并为一个，方便做统一管理（如统一修改样式等）

注意：此操作不可取消，双击嵌套后素材可查看嵌套前的素材分段

时间轴对齐 ​

开启后可帮助我们更好的将两段视频衔接在一起

链接和标记 ​

链接：开启此项后会自动将拖入到这里的音视频链接为一个整体，只可进行整体操作。反之不开启，再拖入视频时会将音频与视频分开，这时就可以单独对音视频操作，如下图所示，这时可以选中素材 “右键”=>“取消链接”这样就看不到帧数差距了（图中素材时间轴左边的“+2.00”字样）

标记：可以在时间轴上打上标记会同步到节目面板上

时间轴工具面板 ​

向前选择 ​

功能：可以选中时间轴前（后）的所有素材

技巧：选中素材按住 “alt” 拖动素材可以快捷复制素材

比例拉伸（倍速和倒放） ​

可以拉长或缩短视频的播放时间，即相当于播放倍率。

也可以点击素材 “右键”=> “速度/持续时间” 做更加精细的调控。比如 2 倍速和倒放

剃刀 ​

可以将视频素材进行切割，分为多个片段

钢笔工具 ​

功能一：可实现视频淡入淡出（透明度由高到低或由低到高）的效果，也可应用在音频上实现某段音频声音由小到大或由大到小

操作：开启钢笔工具在视频开头需要淡入的片段单击打上两个标记再把开头标记往下拖动，即可实现淡入效果。相反在视频结尾操作可实现淡出效果

功能二：画图

操作开启钢笔工具直接在节目面板中画一个图形，时间轴面板会新建出图形素材

文字工具 ​

开启后可直接在节目面板中添加文字

关键帧动画 ​

简单动画 ​

功能：实现图片从右到左移动的动画

操作：拖动时间轴再点击 “位置” 左边图标创建初始位置关键帧，再拖动时间轴（红框圈住的蓝色箭头）并设置需要移动的位置参数即可实现

多段动画 ​

多次拖动时间轴并设置对应运动参数即可实现。可参照如下视频

调节运动路径 ​

功能：可调节运动路径

操作：点击红框然后拖动蓝框中的点即可调节，如果想把路径变为直线（如下图变为直角路径）可点击按住 “ctrl” 点击红框的点即可快捷完成

特效 ​

视频特效 ​

高斯模糊 ​

点击某段视频后在左下角项目面板切换至 “效果” 面板搜索 “高斯模糊” 并双击即可在左上角源面板中调整模糊参数

快捷跟踪模糊 ​

应用场景：如果想把视频中某一部分模糊处理，如果手动一帧一帧处理太过繁琐

操作：先把需要模糊的部分用蒙版工具 “圈出来” 然后调整模糊度再点击 “蒙版路径” 的左右箭头即可快速自动向当前关键帧前后应用模糊度

变换 ​

裁剪 ​

顾名思义是可以将视频按需裁剪

翻转 ​

水平/垂直翻转可以将视频进行左右/上下翻转

图像控制 ​

颜色替换 ​

功能：可以将视频中特定颜色替换成想要的颜色

操作：在效果面板里选择颜色替换，修改目标颜色和替换颜色即可

黑白 ​

可将视频变为黑白色调的视频

扭曲 ​

变形稳定器：当拍摄的视频由于拍摄者自身手部抖动导致视频有抖动不稳的感觉，这时可使用此功能让视频变得 “稳定”。只适用于视频有轻微的抖动，抖动幅度太大则处理后视频效果不佳

边角定位 ​

当我们需要在一段视频的特定区域内播放另一端视频而且需要比较自然的效果可以使用边角定位

放大 ​

可以将视频某一部分放大

杂色与颗粒 ​

杂色 ​

可以制作出老电影的效果

音频特效 ​

室内混响 ​

可以将音频修改为各种场景下的声音效果。可以在预设中选择也可以自行调节下方参数

延迟 ​

可以实现拖音（类似与回音）的效果

音高换挡器 ​

可以调节音频的音调。可以使用预设选择效果也可以操作下方参数自行调整

音频过渡 ​

抠像 ​

亮度键 ​

操作：视频效果=>键控=>亮度键

功能：可以通过亮度信息进行抠像。其中阈值表示抠除深色图像，屏蔽度表示抠除浅色图像。注意这种方式适用于颜色简单且颜色界限分明的情况，如只有黑白两色时

插值遮罩 ​

可以通过计算两个素材的差异进行自动抠像，两个素材中相同的部分抠除，不同的留下

轨道遮罩键 ​

可以将一个视频放入另一个带通道（也可以不带通道）的视频中部分展示

颜色键 ​

当视频背景颜色简单单一可以通过这种方式快速抠像

超级键 ​

适用于比颜色键稍复杂的场景

中间值 ​

可以用来去除水印

字幕 ​

旧版字幕 ​

在软件顶部工具栏依次选择 "文件" => "新建" => "旧版标题"即可打开。在打开的字幕页面即可创建编辑字幕，编辑好后直接点击字幕页面右上角关闭按钮即可，随后在项目面板中就会看到我们刚刚创建的字幕。

新版字幕 ​

在软件顶部工具栏依次选择 "图形" => "新建图层" => "文本"即可直接创建字幕，在右侧 "文本"选项中可调节文字样式，下方可调节文字位置等。

转场 ​

位移转场 ​

笔刷转场 ​

调色技巧 ​

色阶 ​

操作：效果面板 => 视频效果 => 调整 => Levels（色阶）

功能：按下图从上到下顺序

1（Black Input Level）：数值越大视频颜色越暗

2：数值越大视频颜色越亮

3：数值越小视频颜色越亮 对比度越高

4：数值越小视频颜色越暗 对比度越高

快速颜色校正器 ​

可快速对视频进行颜色调整，也可以打关键帧做动画 实现渐变的视频效果

Lumetri 颜色和 Lumetri 范围 ​

基本校正 ​

操作：可在 Pr 软件顶部选择 “窗口”=>"Lumetri 颜色"开启

白平衡：当视频中亮色对比度不佳时可直接点击 “白平衡选择器” 后方的吸管工具直接对视频中白色部分吸取一下即可自动调整

色温：调整视频颜色色调（冷暖色调），双击滑块可还原

创意 ​

Look：表示预设的一些滤镜可直接使用快速调色

强度：调整当前选择的滤镜的强度

淡化胶片：一般用于实现类似回忆的视频效果

锐化：数值越大画面越清晰

自然饱和度&饱和度：饱和度会调整画面整体饱和度，自然饱和度只会调整画面中饱和度较低的对象

曲线 ​

RGB 曲线：用于调整对应颜色（白红绿蓝）亮暗色调曲线

色相饱和度曲线：可以调整画面中某个颜色的饱和度

色轮和匹配 ​

可直接将之前调好色的视频片段或图片直接匹配到另一个视频中
